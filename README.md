# Go Framework Playground
Playground for Go frameworks. Please check the [branches](https://github.com/cylixlee/go-framework-playground/branches) for all demos.

## Demos
Demos of these frameworks below are available on branches of this repo:

- [go-kit](https://github.com/cylixlee/go-framework-playground/tree/go-kit) demonstrates a simple Go Kit server.
- [go-zero/simple](https://github.com/cylixlee/go-framework-playground/tree/go-zero/simple) demonstrates a simple Go Zero API server.
- [go-zero/multiproto](https://github.com/cylixlee/go-framework-playground/tree/go-zero/multiproto) demonstrates a monorepo containing multiple microservices using Go Zero.
